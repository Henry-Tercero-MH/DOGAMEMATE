
# DoGameMate

Proyecto React inicializado con Vite.

## Estructura principal
- **src/components/**: Componentes reutilizables
- **src/hooks/**: Custom hooks
- **src/context/**: Contextos de React
- **src/config/**: Configuración y constantes
- **src/pages/**: Vistas o páginas principales
- **src/assets/**: Imágenes, fuentes, etc.
- **src/utils/**: Utilidades y helpers
- **src/tests/**: Pruebas unitarias

## Scripts útiles
- `npm run dev`: Iniciar servidor de desarrollo
- `npm run build`: Generar build de producción
- `npm run lint`: Ejecutar ESLint
- `npm run test`: Ejecutar pruebas (Jest)

## Herramientas incluidas
- Vite
- React
- ESLint + Prettier
- Jest + Testing Library
